# 📱 מדריך בניית APK — איילון ביטוח

## מה תקבל בסוף?
קובץ `.apk` שמותקן ישירות על הטלפון שלך.  
האפליקציה: מזינים ת"ז ← מקבלים SMS ← הקוד מוזן אוטומטית ← רואים את הפוליסות.

---

## שלב 1: פתח חשבון GitHub (חינמי)
1. כנס ל-[https://github.com](https://github.com)
2. לחץ **Sign up** ← צור חשבון חינמי

---

## שלב 2: צור Repository חדש
1. לחץ על **+** (פינה ימנית עליונה) ← **New repository**
2. שם: `ayalon-app`
3. בחר **Public** (נדרש לזמן בנייה חינמי)
4. לחץ **Create repository**

---

## שלב 3: העלה את הקבצים

### אפשרות A — דרך האתר (הכי קל):
לחץ **uploading an existing file** בדף ה-repository החדש ולאחר מכן גרור את שלושת הקבצים:

```
ayalon-app/
├── buildozer_project/
│   ├── main.py          ← קוד האפליקציה
│   └── buildozer.spec   ← הגדרות הבנייה
└── .github/
    └── workflows/
        └── build.yml    ← הוראות הבנייה האוטומטית
```

> ⚠️ חשוב: שמור על מבנה התיקיות בדיוק כפי שמופיע למעלה

### אפשרות B — דרך Git (אם מותקן על המחשב):
```bash
git clone https://github.com/YOUR_USERNAME/ayalon-app
cd ayalon-app
# העתק את הקבצים
git add .
git commit -m "Initial commit"
git push
```

---

## שלב 4: הפעל את הבנייה
1. כנס ל-Repository שלך ב-GitHub
2. לחץ על לשונית **Actions**
3. תראה **Build Ayalon APK** — לחץ עליו
4. לחץ **Run workflow** ← **Run workflow** (כחול)
5. המתן **~15-25 דקות** לסיום הבנייה

---

## שלב 5: הורד את ה-APK
1. כשהבנייה מסתיימת (✅ ירוק) — לחץ עליה
2. למטה בעמוד תראה **Artifacts**
3. לחץ על **ayalon-apk** ← הורד ZIP
4. פתח את ה-ZIP ← תמצא קובץ `*.apk`

---

## שלב 6: התקן על הטלפון
1. העבר את קובץ ה-APK לטלפון (WhatsApp, Drive, כבל USB)
2. פתח אותו בטלפון
3. Android יבקש לאשר התקנה ממקור לא ידוע:
   - **הגדרות** ← **אבטחה** ← **מקורות לא ידועים** ← **אפשר**
4. התקן!

---

## שלב 7: הפעל את האפליקציה
1. פתח את **איילון פוליסות**
2. תן הרשאת SMS כשנשאל
3. הזן תעודת זהות ← לחץ כניסה
4. קוד ה-SMS יתמלא **אוטומטית**
5. רואה את הפוליסות 🎉

---

## אם הבנייה נכשלת
לחץ על הבנייה הכושלת ← פתח **build** ← גלול לשורות האחרונות ← שלח לי צילום מסך ואתקן.

---

## זמן בנייה בפעם הראשונה
הפעם הראשונה לוקחת ~25 דקות כי GitHub מוריד את ה-Android SDK.  
בניות הבאות: ~5 דקות בזכות ה-cache.
